# WellTracker

건강 추적 애플리케이션 - 혈압과 혈당을 기록하고 관리하세요.

## 기능

- 📊 혈압 및 혈당 기록 및 추적
- 📈 시각적 차트로 건강 데이터 분석
- 💾 로컬 저장소 자동 저장
- 📁 JSON 파일로 데이터 내보내기/가져오기
- 🤖 AI 기반 건강 인사이트 (Gemini API)

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 앱을 확인하세요.

### 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist/` 디렉토리에 생성됩니다.

## 기술 스택

- **프레임워크**: React 19
- **빌드 도구**: Vite 6
- **차트 라이브러리**: Recharts 3
- **아이콘**: Lucide React
- **스타일링**: Tailwind CSS (CDN)
- **백엔드**: Firebase 11
